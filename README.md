## **Overview**

This project is a SwiftUI iOS app built in Xcode. All images and assets required to run the app are already included in the project’s **Assets.xcassets** folder.

## **Requirements**

* **macOS**
* **Xcode** (any recent version that supports SwiftUI)
* **iOS Simulator** (included with Xcode)


## **How to Run the App**

### **1. Clone the repository**

```bash
git clone https://github.com/GalNi1/iOSCardGame.git
cd iOSCardGame
```

### **2. Open the project in Xcode**

Double-click the `.xcodeproj` or `.xcworkspace` file.

### **3. Select a device**

In the top bar of Xcode:

* Choose an **iPhone simulator** (e.g., iPhone 15, iPhone 15 Pro)

### **4. Run**

Press **Cmd + R** or click the **Run ▶️** button.

Xcode will compile the project and launch it automatically in the selected simulator.
All included images will load correctly through the asset catalog, no extra configuration needed.
